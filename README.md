# CCE 102 — Stewardship and the Built Environment

Course schedule and topic plan for **CCE 102** (BYU Civil & Construction Engineering, Fall 2026).

## What this is

A single-page, BYU-branded course schedule that reads its data live from a published Google Sheet. Students and faculty can use it to see the topic arc, what's assigned, estimated time-to-complete, and project benchmarks, color-coded by chapter.

## How it works

`index.html` is a self-contained page (HTML, CSS, JS — no build step). On every load it fetches the published-CSV endpoint of the planning sheet and renders the schedule. To update the course content, edit the Sheet — the page picks up changes automatically (within Google's ~5-minute publish cache).

## Brand

Header lockup is the official BYU Civil and Construction Engineering sub-brand lockup (horizontal, 2-line, white), used as provided per the BYU brand guide. Colors and typography (IBM Plex Sans) follow `brand.byu.edu`.
